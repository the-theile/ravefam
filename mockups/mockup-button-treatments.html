Button treatments

10 ways to move the primary CTA off hot pink

Same pill shape, weight, and glow language as the current button — only the color and surface treatment change. Live buttons below, hover each one to see the interaction.

Current Pink → purple gradient, used for every primary action site-wide.
01Electric Cyan

Solid cyan, dark ink text. Pulls from the app's secondary neon instead of introducing a new hue — reads as "signal," calmer than pink but still electric.

02Voltage Gradient

Cyan into acid green, the same diagonal sweep as today's pink→purple. Feels like a lighting rig cross-fade rather than a brand swap.

03Acid Green Solid

The green already anchors the wordmark and the "live" eyebrow dot — this makes the CTA agree with the logo instead of competing with it.

04Yellow Flash

Glow-stick yellow on black. Highest urgency and contrast of the set — good if the CTA needs to win against a busy photo background.

05Holographic

Cyan → purple → green, the exact gradient already used for the hero headline text. Hover slides the gradient like a laser sweep.

06Outline Neon

Transparent fill, cyan hairline, fills in on hover. Lightest-weight option — lets a photo or dark hero breathe behind it.

07Frosted Glass

Matches the blur already used on the nav bar and the save-to-home-screen modal, so the CTA feels native to the rest of the chrome instead of shouting over it.

08Duotone Split

Hard-edge cyan/purple split instead of a blend — a wristband-tab look. More graphic, less "gradient button" than the current default.

09Chrome Rig

Dark brushed-metal fill with a cyan edge-light, like backstage truss hardware. Reads premium and moodier than any of the flat neons.

10Minimal Mono

Off-white solid, black text, no color at all. The safest, most accessible option — lets the neon blobs and photography carry all the color instead.

All ten keep the existing pill radius (100px), Outfit 800 weight, and glow-shadow pattern from .btn-hero-primary — swap in whichever wins and the rest of the button system (hover lift, focus ring, disabled state) carries over unchanged.
