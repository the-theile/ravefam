Design Review

Vibe Tag Picker — 4 Redesigns

Four different approaches to selecting/adding vibe tags while editing a profile, replacing the current flat 48-pill "wall of tags." Each mockup is fully clickable — search, filter, toggle, swipe, expand.

Mockup 1
Search + Filter + Sticky Tray
Smallest change from today's UI: adds search, category filters, and a pinned "your vibes" tray on top of the existing pill cloud.
Mockup 2
Bottom-Sheet Modal
Moves the picker into a dedicated full-height sheet, decluttering the main profile-edit page down to a summary + "Edit" button.
Mockup 3
Swipeable Discovery Deck
Playful Tinder-style one-card-at-a-time browsing for discovering new tags, with a compact list view for quick edits.
Mockup 4
Grouped Accordion
Buckets all 48 tags into collapsible themed categories with per-section selected counts; search auto-expands and highlights matches.
